开放查询

您可以直接访问 https://www.mcbl.cc/list.json 获得 JSON 格式的名单列表，然后根据 id 字段读取上榜原因。

上榜原因读取：https://www.mcbl.cc/text/<id>.md

例如 ID 为 123，请求的 URL 就是 https://www.mcbl.cc/text/123.md

上榜原因列表返回的文本内容为 Markdown 格式。
